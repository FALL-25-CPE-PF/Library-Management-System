Features


Admin

Login with default credentials (admin/admin123)

Add Student with type (UG, PG, RS, GUEST) and password

Add Book with ID, title, author, and quantity

View Students (all registered students)

View Books (all books with quantity)

Approve Book Requests submitted by students before issuing

Change Password with confirmation


Student

Login with student ID and password

View Available Books

Request Book (requires admin approval before issuing)


Return Book

Automatically calculates fine if returned after 7 days

Shows rent, fine, and total amount

View Fine for all returned books

Change Password securely


File Handling

All files are automatically created if missing:

admin.csv, students.csv, books.csv, requests.csv, issued_books.csv, fine.csv

Admin and student passwords can be changed with validation


How to Run

Ensure Python 3 is installed.

Download or clone the repository.

Navigate to the project folder in terminal.

Run the program:        ===  python main.py

Follow the console menu to login as Admin or Student.
